1. Pick a task card.
2. Read its **Goal**, **Files**, **Depends on**, and **Acceptance**.
3. Follow the [AGENT_GUIDE.md](../AGENT_GUIDE.md) workflow.
4. Mark the task as complete and update the card with the completion date.
